Data Types in Python

Data type defines what kind of value the variable holds. Example: number, decimal, text, true/false.

Data Type Example
int 10
float 10.5
complex 2+3j
str "Hello"
bool True
list [1,2,3]
tuple (1,2,3)
set {1,2,3}
dict {"a":1}
int (Integer)

int stores whole numbers (no decimals).

age = 21
print(type(age))
Output: <class 'int'>
float (Decimal)

float stores numbers with decimal points.

price = 99.5
print(type(price))
Output: <class 'float'>
complex (Complex Number)

complex stores numbers with real + imaginary (j).

num = 2+3j
print(type(num))
Output: <class 'complex'>
str (String)

str stores text inside quotes.

name = "Python"
print(type(name))
Output: <class 'str'>
bool (Boolean)

bool stores True or False.

is_student = True
print(type(is_student))
Output: <class 'bool'>
list (List)

list stores multiple values (mutable).

nums = [1,2,3]
print(type(nums))
Output: <class 'list'>
tuple (Tuple)

tuple stores multiple values (immutable).

t = (1,2,3)
print(type(t))
Output: <class 'tuple'>
set (Set)

set stores unique values.

s = {1,2,3}
print(type(s))
Output: <class 'set'>
dict (Dictionary)

dict stores data in key-value pairs.

d = {"a":1}
print(type(d))
Output: <class 'dict'>

Type Conversion

Changing one data type into another type.

Implicit Conversion

Python automatically converts type when needed.

a = 5
b = 2.5
c = a + b
print(type(c))
Output: <class 'float'>
Explicit Conversion

You manually convert the type using functions.

x = "10"
y = int(x)
print(type(y))
Output: <class 'int'>
